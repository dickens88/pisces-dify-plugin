# Privacy Policy

The Pisces plugin connects exclusively to the Pisces API server configured by the user (via the `base_url` credential). It does not collect, store, or transmit any data to third-party services.

All IOC data and incident information is sent only to the user's own self-hosted Pisces backend. Credentials (JWT token) are stored securely within Dify's credential management and are never logged or shared.
